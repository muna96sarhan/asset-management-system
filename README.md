# نظام إدارة الأصول والعهود (Asset Management System)

مشروع متكامل للباك إند (Back-end) مبني باستخدام تقنيات حديثة لإدارة الأصول والعهود داخل الشركات والمؤسسات وربطها بالموظفين.

## التقنيات المستخدمة (Technologies)
- **Java** (إصدار 17 أو أحدث)
- **Spring Boot** (إطار العمل الأساسي)
- **Spring Data JPA** (للتعامل مع قاعدة البيانات)
- **MySQL Database** (لتخزين البيانات)
- **Maven** (لإدارة إعدادات ومكتبات المشروع)

## محتويات المشروع
- كود الجافا كاملاً والـ APIs الخاصة بـ (Assets).
- ملف السكربت الكامل لقاعدة البيانات والجداول والـ Stored Procedures باسم `asset_management_db.sql`.
